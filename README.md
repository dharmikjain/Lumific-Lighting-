# Lumific Lighting

Premium architectural lighting platform with AI-powered tools, product configurator, and smart quotation engine.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server (http://localhost:3000)
npm run dev

# Build for production
npm run build

# Start production server
npm run preview
```

## Project Structure

```
Lumific/
├── my-website/          # Next.js app (main site)
│   ├── src/app/         # Pages & routes
│   ├── src/components/  # Shared components
│   └── ...
└── package.json         # Root scripts (delegates to my-website)
```
